## Configuration

| Key          | Values                                          |
|--------------|-------------------------------------------------|
| notifier     | Must be `discord`                               |
| url          | Webhook URL                                     |
| author       | Username to appear in slack. (optional)         |
| author_image | Emoji code to use for user's avatar. (optional) |
